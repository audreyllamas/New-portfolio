AUDREY LLAMAS PORTFOLIO WEBSITE

HOW TO OPEN
1. Extract the ZIP file.
2. Open index.html in Chrome, Edge, or Firefox.
3. Keep the folder structure unchanged so the images and Excel downloads work.

IMPORTANT BEFORE SENDING TO AN EMPLOYER
1. Replace the three clearly marked graphic design placeholders inside the assets folder:
   - graphic-design-1.svg
   - graphic-design-2.svg
   - graphic-design-3.svg
   You may keep the same filenames, or update the image paths in index.html.
2. Add your real LinkedIn link only when the profile is complete.
3. Review and understand every Excel workbook before presenting it. The workbooks use fictional sample data and are provided as editable portfolio projects.
4. Never upload real patient names, photographs, contact information, treatment records, or clinic-identifying information to the public portfolio.
5. Confirm that “President’s Lister and Dean’s Lister” is accurate and supported by your school records.

EXCEL FILES INCLUDED
- downloads/Aftersales_Service_Tracker.xlsx
- downloads/Cafe_Business_Tracker.xlsx
- downloads/Student_Gradebook_Dashboard.xlsx

HOW TO EDIT TEXT
Open index.html using VS Code or any text editor. Search for the sentence or heading you want to replace, then save the file.

HOW TO PUBLISH FREE
You can upload the entire folder to GitHub Pages, Netlify, or another static website host. The site does not require a database or paid software.
